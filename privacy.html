Privacy Policy

Last updated: April 2026

1. Data Collection

This application collects only the data necessary to provide its services: OAuth tokens for YouTube and TikTok API access.

2. Data Storage

All data is stored locally on your machine. No data is sent to third-party servers except YouTube and TikTok APIs for video publishing.

3. Third-Party Services

This app integrates with:

  • YouTube Data API (Google)
  • TikTok Content Posting API
  • Gemini AI (Google) for content generation
  • Pollinations AI for image generation

4. Data Sharing

We do not sell or share your data. API tokens are stored locally and used only for authorized API calls.

5. Data Deletion

You can delete all stored data by removing the credentials folder and clearing the application cache.

6. Contact

For privacy concerns, open an issue on the project repository.
